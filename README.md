# MyCar+ Web V5.4

## Νέα στο V5.4
- Εγκατάσταση σαν εφαρμογή (PWA)
- Offline λειτουργία με Service Worker
- Κουμπί εγκατάστασης σε υποστηριζόμενους browsers
- Οδηγία εγκατάστασης για iPhone
- Εξαγωγή πλήρους backup σε JSON
- Εισαγωγή και επαναφορά backup
- Ένδειξη online / offline κατάστασης
- Επιπλέον βελτιώσεις για κινητά
- Διατηρούνται όλες οι λειτουργίες του V5.3
